kiama 0.9.0 API

This document is the API specification for kiama 0.9.0 API

Class Summary
trait CharPackratParsers extends PackratParsers with CharParsers
Packrat parsers that operate on character input.
trait CharParsers extends Parsers
Parsers that operate on character input.
trait PackratParsers extends Parsers
Parsers that use the packrat parsing approach to memoise parsing results, including support for left recursive grammar rules. The algorithms used here are from "Packrat parsers can support left recursion" by Warth, Douglass and Millstein, ACM SIGPLAN Symposium on Partial Evaluation and Semantics-based Program Manipulation, 2008.
trait Parsers extends AnyRef
Parser combinator library modelled on the Scala parser combinator library, and through it on various versions of parser combinator libraries for Haskell.
